NJIT Capstone Project

During the NJIT Capstone Project during Spring 2023, I worked in a group with 4 other people in order to create a website for the entity WillLouden, INC. This website, pyrepay.com, was designed as a fincancial service to allow a user to create an account and log into that account. Following this, the user can connect their pyrepay to another website named turntswag which is in charge of connecting to their bank. Once connected, Pyrepay will allow the user to send, request, and recieve money along with seeing recent transactions and adding contancts. To complete this project, I utilized HTML for the forms, CSS for the styling, PHP and SQL to connect to the databases required.
