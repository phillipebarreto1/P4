Android Watch Store

For a class, I designed basic app that loads a list of watches from the web, which the app is then able to display. With the list loaded, a user is able to add a new watch to the list, show specific details of a watch, remove a watch from the list, show the average price of all dive watches, show the number of automatic watches, and show all watches of a given brand. To complete this project, I used Java and some basic XML. With Java, I used injection and singleton methods along with other classes.
