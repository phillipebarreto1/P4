Android Server Calculator

This project required 2 seperate programs to run simultaneously. The first part was to program a server that an android app would socket to. In the server, there are defined methods that act as operations for big numbers.

For the second part of the project, the client will ask a user to input the domain and port # to connect to the main activity where it will then prompt the user to use a specific operation and two numbers. The client will send those three lines to the server and use them to calculate that operation. Finally, the server will send back the corrrect answer or validation error to the client.
